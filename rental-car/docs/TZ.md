# Мини-ТЗ: Сайт аренды автомобилей (CarRent Pro)

## 1. Название
CarRent Pro - Сайт для Аренды автомобилей

## 2. Цель
Разработка консольного приложения для управления процессом аренды автомобилей.

## 3. Основные функции
- Просмотр каталога автомобилей
- Поиск и фильтрация
- Бронирование авто
- Управление автопарком (админ)

## 4. Технологии
- C# .NET 8.0
- Консольный интерфейс
- JSON для хранения данных
