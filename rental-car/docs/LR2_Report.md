# Отчёт ЛР2 — Система аренды автомобилей

## Цель работы
Разработка системы управления арендой автомобилей с использованием архитектуры 
с разделением на слои (App и Core).

## Что сделано
1. Создана структура проекта:
   - Program.cs - консольное приложение с меню
   - CarRental.Core - ( Еще не сделанно( )

2. Реализованы модели данных:
   - Car - автомобиль (марка, модель, год, цена аренды, статус)
   - Rental - аренда (клиент, даты, автомобиль, стоимость)
   - RentalStatus - статусы аренды (активна, завершена, отменена)

3. Реализованы сервисы:
   - CarService - управление автомобилями (добавление, поиск, обновление)
   - RentalService - управление арендой (оформление, проверка доступности)

4. Реализованы функции:
   - Добавление автомобилей в парк
   - Просмотр всех автомобилей
   - Поиск свободных автомобилей на указанные даты
   - Оформление аренды с расчетом стоимости
   - Просмотр активных аренд
