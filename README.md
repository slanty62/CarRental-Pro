## 🎯 Описание проекта

CarRental Pro - консольное приложение для управления арендой автомобилей. Проект разработан в рамках учебной лабораторной работы с применением принципов чистой архитектуры.

## ✨ Функционал ✅

### 🚗 Управление автомобилями
- ✅ Просмотр всех автомобилей в парке
- ✅ Добавление новых автомобилей
- ✅ Поиск по марке/модели
- ✅ Фильтрация по цене
- ✅ Отображение статуса (свободен/занят)

### 📅 Бронирование
- ✅ Проверка доступности на даты
- ✅ Оформление аренды
- ✅ Расчет стоимости
- ✅ Ввод данных клиента

### 👨‍💼 Администрирование
- ✅ Панель администратора
- ✅ Просмотр бронирований
- ✅ Статистика
- ✅ Управление авто

### Установка
```bash
# 1. Клонируйте репозиторий
git clone https://github.com/slanty62/CarRental.git
cd CarRental

# 2. Восстановите зависимости
dotnet restore

# 3. Запустите приложение
dotnet run --project CarRental.App
